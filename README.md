개발툴로 XCode를, 개발언어는 SwiftUI를 사용했습니다.   
Tabview를 통해 기본적인 UI와 화면전환기능을 구현했습니다.   
Firebase의 Firestore Database와 연동하여 게시판의 CRUD기능을 구현했습니다.   
Firebase와 연동하여 구글아이디로 로그인 기능을 구현했습니다.
